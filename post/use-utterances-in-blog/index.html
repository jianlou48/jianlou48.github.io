Security Problem

I had use a lot of comments widget built on Github issues,like gitment,gittalk,even a simple javascript jump to the issues page.They all have a problem before security and convenience.

The problem’s source is the access api of Github.It is too dangerous to ask for the whole account and respositories’s access to make a comments.If you do that,maybe nobody like to comment to your posts.And that is not polite to your visitor.

utterances works very good.It will want these access when the first time you want to comments:

  • Your account: Verify your GitHub account
  • Resources: Determine what resources both you and utterances can access

Imporant Configuration

You can install utterances app and see your settings here.

Permissions:

  • No access to code
  • Read access to metadata
  • Read and write access to issues

You can configure repository access for only select repositories.That’s nice!
